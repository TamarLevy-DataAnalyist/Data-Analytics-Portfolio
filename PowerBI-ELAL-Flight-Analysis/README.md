# EL AL Flight Analysis Dashboard

## Project Overview
Power BI dashboard analyzing EL AL flight data, including revenue, passengers, cancellations, flight activity, and occupancy rates.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

## Main Features
- Interactive dashboards
- Drill Through
- Bookmarks
- Slicers
- Mobile Layout

## Key Insights
- Revenue varies by destination and month.
- Occupancy rates differ between selected destinations.
- Cancellation patterns change over time.
